## 🎯 安装顺序推荐

### 第一批：核心基础（必装）

```bash
# 搜索和知识基础
openclaw skill install tavily-search
openclaw skill install memory-lancedb-pro

# 多Agent协作基础
openclaw skill install multi-agent-pipeline
```

**原因**: 这些是系统运行的基础依赖

---

### 第二批：内容创作（根据需求）

```bash
# 公众号
openclaw skill install wechat-article-pro

# 小红书
openclaw skill install redbook-maker

# 通用优化
openclaw skill install humanizer
```

**原因**: 根据你运营的平台选择

---

### 第三批：高级功能（按需）

```bash
# Agent自主性
openclaw skill install agent-autonomy-kit

# 知识图谱
openclaw skill install knowledge-graph

# 自动化
openclaw skill install workflow-automator
```

**原因**: 提升系统智能化程度

## 🔧 配置建议

### 敏感配置

部分Skills需要API密钥：

| Skill | 需要的Key | 配置位置 |
|-------|-----------|----------|
| tavily-search | Tavily API Key | `~/.openclaw/skills/tavily-search/config.json` |
| memory-lancedb-pro | LanceDB配置 | `~/.openclaw/memory/` |

### 安装后检查

安装后测试：

```bash
# 列出已安装Skills
openclaw skill list

# 查看Skill详情
openclaw skill info <skill-name>

# 测试Skill
openclaw skill test <skill-name>
```

## 🆘 常见问题

### Q: Skill安装失败？

```bash
# 检查网络连接
ping clawhub.ai

# 检查权限
openclaw doctor

# 查看详细错误
openclaw skill install <skill-name> --verbose
```

### Q: Skill无法使用？

```bash
# 检查是否启用
openclaw skill enable <skill-name>

# 查看日志
tail -f ~/.openclaw/gateway.log | grep <skill-name>
```

### Q: 不知道该装哪些？

按以下顺序：
1. 先装核心基础（tavily-search + memory-lancedb-pro）
2. 再装你需要的创作类Skill
3. 最后装自动化和高级功能
