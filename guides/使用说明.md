## 📖 两个版本

### 🚀 版本A：Claude Code版（推荐给学员）

**适合人群**：
- ✅ 电脑上已安装Claude Code
- ✅ 不想手动编辑JSON文件
- ✅ 想快速完成搭建

**使用方式**：
1. 打开 `README-ClaudeCode版.md`
2. 复制 `快速开始-给ClaudeCode发送这条.md` 中的指令
3. 粘贴到Claude Code对话框
4. 等待Claude Code自动完成所有配置

**特点**：
- 一条指令搞定全部配置
- Claude Code自动创建文件、编辑JSON
- 适合零基础小白

---

### 🛠️ 版本B：手动配置版

**适合人群**：
- ⚠️ 没有Claude Code
- ⚠️ 想了解每个配置细节
- ⚠️ 需要深度定制

**使用方式**：
1. 阅读 `02-零基础上手指南.md` 了解环境准备
2. 阅读 `03-Agent配置详解.md` 创建Agent
3. 阅读 `04-Telegram配对流程.md` 配置Bot
4. 使用 `05-完成检查清单.md` 验证配置

**特点**：
- 详细说明每个配置项
- 提供完整JSON示例
- 适合有一定技术基础的用户

## 🆘 遇到问题？

### 问题1：不知道选哪个版本

**回答**：
- 如果你是学员，已安装Claude Code → 选版本A
- 如果你想学习配置细节 → 选版本B

### 问题2：Claude Code执行失败

**回答**：
- 检查Claude Code是否有终端权限
- 尝试手动执行第一步：`npm install -g openclaw`
- 把错误信息发给Claude Code，让它帮忙排查

### 问题3：配置完成后Bot不回复

**回答**：
- 检查Gateway状态：`openclaw gateway status`
- 查看日志：`tail -f ~/.openclaw/gateway.log`
- 重启Gateway：`openclaw gateway restart`

### 问题4：想了解更多原理

**回答**：
- 阅读 `01-系统架构总览.md`
- 参考原始架构文档：`../星河AI-Agent-Teams-V2.5.md`
- 查看Agent配置示例
