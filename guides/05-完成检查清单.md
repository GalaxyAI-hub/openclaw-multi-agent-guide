## 🆘 遇到问题？

### 常见问题速查

**Gateway启动失败**
- 检查端口占用: `lsof -i :18790`
- 检查JSON语法: `python3 -m json.tool ~/.openclaw/openclaw.json`
- 查看日志: `tail -f ~/.openclaw/gateway.log`

**Bot不回复**
- 确认Gateway运行中
- 确认Bot Token正确
- 确认已执行 `/pair`
- 重启Gateway: `openclaw gateway restart`

**Agent间无法通信**
- 检查 `sessions.visibility` = "all"
- 检查 `agentToAgent.enabled` = true
- 检查Agent ID在 `allow` 列表中
- 重启Gateway

**配置不生效**
- 修改openclaw.json后必须重启Gateway
- 使用 `openclaw gateway restart`
